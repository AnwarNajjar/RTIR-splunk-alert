## Release Notes

##### Version 1.0.0 March 26 2020
##### RTIR Alert for Splunk Enterprise Security 1.0.0

## About RTIR Alert for Splunk Enterprise Security

The RTIR Alert for Splunk Enterprise Security allow Splunk ES users to open a ticket with IR team on [RTIR platform](https://bestpractical.com/rtir) within the ES adaptive respone actions. 

Note: it's written in python3 and it's compatible with ES v5.0.1+ and splunk enterprise v7.0.0+. 

## Requirements

- RTIR platfrom 4.2.2+
- Splunk ES v5.0.1+
